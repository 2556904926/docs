## 📖 理论 —— 先懂原理，再谈调参

### 第一步：认识“系统”到底是什么

- **[控制系统建模基础](./1_理论/控制系统建模基础/readme.md)**  
  从一阶 RC 到高阶振荡环节，教你用**主导极点**一眼看穿系统本质。  
  *关键收获：给你一个传递函数，你能立刻说出它“大概会怎么动”。*

- **[频域分析方法](./1_理论/频域分析方法/readme.md)**  
  伯德图不只是考试题——**PM 决定抖动，带宽决定响应速度，谐振峰决定极限性能**。  
  *关键收获：看懂开环伯德图，就能预判闭环阶跃响应。*

### 第二步：用控制器“改造”系统

- **[PID控制原理与整定](./1_理论/PID控制原理与整定/readme.md)**  
  Z-N 法、Cohen-Coon 法，以及更实用的**工程整定经验**。不绕弯子，直接告诉你怎么调。

- **[超前滞后校正](./1_理论/超前滞后校正/readme.md)**  
  当 PID 不够用时，用零极点搬移来**拓展带宽或压制谐振**。

- **[滤波器设计](./1_理论/滤波器设计/readme.md)**  
  低通滤噪声、高通滤漂移、陷波器专治机械谐振——**控制器的“外科手术工具”**。

- **[观测器设计](./1_理论/观测器设计/readme.md)**  
  状态不可测？用全维/降维观测器**重构状态**，为高级控制铺路。

- **[离散控制系统](./1_理论/离散控制系统/readme.md)**  
  数字控制器的必修课：Z 变换、Tustin 离散化、防积分饱和的数字实现。

> 💡 **引导建议**：如果你是初学者，请严格按“建模基础 → 频域分析 → PID/校正 → 离散化”顺序阅读。后面的观测器可以等遇到状态反馈需求时再回看。

## 💻 仿真 —— 动手验证，拒绝纸上谈兵

- **[Simulink仿真模板](./2_仿真/Simulink仿真模板/readme.md)**  
  一个**拿来就能用**的通用仿真框架，包含信号生成、被控对象、控制器、示波器四件套。  
  *使用场景：拿到新控制算法，10 分钟内搭好验证环境。*

- **[仿真脚本模板](./2_仿真/仿真脚本模板/readme.md)**  
  批量跑参数扫描、自动保存数据、生成对比曲线——**让你告别手动改参数跑仿真**。

## 🧩 代码 —— 让算法真正“跑起来”

- **[MATLAB通用函数库](./3_代码/MATLAB通用函数库/readme.md)**  
  对 `tf/bode/step/pidtune` 的二次封装，让你**三行代码画出开环/闭环伯德图**，五行完成 PID 预整定。

- **[C嵌入式实现](./3_代码/C嵌入式实现/readme.md)**  
  将设计好的数字控制器**翻译成 C 代码**，包含定时器中断执行框架、抗积分饱和处理。  
  *关键提醒：这里不是简单翻译，而是考虑了量化误差和计算延迟的工程化实现。*

## 🔬 工程 —— 三个真实被控对象，手把手带你走完闭环

> 这是本库的**精华实战区**，每个案例都包含：**物理建模 → 控制器设计 → 仿真验证 → 实验数据对比**。

- **[音圈电机控制](./4_工程/音圈电机控制/readme.md)**  
  高频响、小行程的典型代表。你会学到如何**用陷波器压制机械谐振**，实现 100Hz 以上带宽。

- **[压电陶瓷控制](./4_工程/压电陶瓷控制/readme.md)**  
  迟滞非线性是最大敌人。这里提供**逆迟滞补偿 + PID 闭环**的组合方案。

- **[BLDC控制](./4_工程/BLDC控制/readme.md)**  
  从电机模型到 FOC（磁场定向控制），再到速度环/电流环参数整定——**一套完整的伺服方案**。

> 💡 **引导建议**：选一个和你当前项目最接近的案例，先完整复现一遍仿真，再尝试替换成你自己的被控对象模型。
